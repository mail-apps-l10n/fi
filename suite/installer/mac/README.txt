  = = = = = = = = = = = = SeaMonkeyn Lueminut-tiedosto = = = = = = = = = = = =

  Kiitos, kun latasit SeaMonkeyn. Tämä tiedosto sisältää tietoja seuraavista
  asioista:
  
  - Järjestelmä- ja laitteistovaatimukset
  - Asentaminen
  - Mihin tiedot tallennetaan
  - Asennuksen poistaminen
  - SeaMonkeyn hankkiminen
  - Lisätiedot ja palaute
  - Lisenssi

  Jos haluat tietoja tunnetuista käyttö- ja asennusongelmista, lue
  julkaisutiedot. Ne löytyvät itse julkaisujen kanssa osoitteesta
  
  http://www.seamonkey-project.org/releases/.


  Järjestelmä- ja laitteistovaatimukset
  -------------------------------------

  Järjestelmävaatimukset:

  - Mac OS X 10.5 tai uudempi


  Hardware Requirements:

  - Macintosh-tietokone Intel x86- tai PowerPC G4/G5 -suorittimella
  - 200 Mt vapaata kiintolevytilaa


  Asentaminen
  -----------

  Tee seuraavasti asentaaksesi SeaMonkeyn:

  1. Vedä SeaMonkey-sovelluskuvake Sovellukset-kansioon

  2. Poista levykuva

  3. Jos haluat päästä SeaMonkeyhin käsiksi Dockista, vedä SeaMonkey-kuvake
     Sovellukset-kansiosta Dockiin


  Mihin tiedot tallennetaan
  -------------------------

  Kun käynnistät SeaMonkeyn ensimmäisen kerran, SeaMonkey-kansio luodaan
  kohteeseen

  ~/Library/Application Support/ (jossa ~ on kotihakemistosi).

  Asetukset, kirjanmerkit, sähköpostiviestit, salasanat yms. tallennetaan
  SeaMonkey-kansion sisällä olevaan profiilikansioon. Profiili sijaitsee
  kohteessa

  ~/Library/Application Support/SeaMonkey/Profiles/[randomString.profileName].

  Väliaikaistiedostot tallennetaan kohteeseen ~/Library/Cache/SeaMonkey.
  Tietyt asetukset (kuten Tallenna-dialogin viimeiseksi käyttämä juurikansio)
  tallennetaan kohteeseen

  ~/Library/Preferences/org.mozilla.seamonkey.plist.


  Asennuksen poistaminen
  --------------

  Jos haluat poistaa SeaMonkeyn, mutta pitää asetukset, kirjanmerkit yms.,
  vedä SeaMonkey-kuvake Sovellukset-kansiosta Roskakoriin.

  Jos haluat poistaa kaikki SeaMonkeyhin liittyvät tiedostot tietokoneeltasi
  (sisältäen asetukset, kirjanmerkit, sähköpostiviestit, salasanat yms.)
  tee seuraavasti:

  1. Vedä SeaMonkey-kuvake Sovellukset-kansiosta Roskakoriin

  2. Vedä kansio ~/Library/Application Support/SeaMonkey Roskakoriin

  3. Vedä kansio ~/Library/Cache/SeaMonkey Roskakoriin

  4. Vedä tiedosto ~/Library/Preferences/org.mozilla.seamonkey.plist
     Roskakoriin


  SeaMonkeyn hankkiminen
  ----------------------

  SeaMonkeystä tehdään säännöllisesti yöllisiä koosteita testausta varten.
  Ne ovat kehitysversioita ja saattavat olla bugisia.

  Yölliset koosteet ovat saatavilla mozilla.orgin FTP-sivustolla osoitteessa
  http://ftp.mozilla.org/pub/mozilla.org/seamonkey/nightly/

  Jos haluat aivan tuliterän koosteen, katso

  http://www.seamonkey-project.org/dev/nightly

  Jos etsit viimeistellympää versiota SeaMonkeystä, voit ladata virallisen
  julkaisun osoitteesta

  http://www.seamonkey-project.org/releases/

  Lisätiedot ja palaute
  ---------------------

  Lisätietoja SeaMonkeystä saat osoitteesta http://www.seamonkey.org/
  Jos haluat lähettää virheraportteja tai muuta palautetta, katso
  http://www.seamonkey-project.org/dev/get-involved ja tutustu Bugzillaan
  osoitteessa https://bugzilla.mozilla.org, mistä löydät tunnetut virheet, 
  virheraporttien kirjoitusohjeet yms. Voit myös saada ohjeita Bugzillan
  käyttöön IRC-kanavalta #mozillazine palvelimella irc.mozilla.org.


  Lisenssi
  -------

  SeaMonkey on osoitteessa
  http://www.mozilla.org/legal/eula/seamonkey-en.html olevassa
  lisenssisopimuksessa olevien ehtojen alainen.
