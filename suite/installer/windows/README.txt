= = = = = = = = = = =  SeaMonkeyn Lueminut-tiedosto  = = = = = = = = = = =

==========================================================================

SeaMonkey on sen mukana tulleen lisenssisopimuksen ehtojen alainen.

Tässä Lueminut-tiedostossa on tiedot järjestelmävaatimuksista sekä
Windows-version asennusohjeet.

Lisätietoja SeaMonkeystä saat osoitteesta http://www.seamonkey.org/
Jos haluat lähettää virheraportteja tai muuta palautetta, katso
http://www.seamonkey-project.org/dev/get-involved ja tutustu Bugzillaan
osoitteessa https://bugzilla.mozilla.org, mistä löydät tunnetut virheet, 
virheraporttien kirjoitusohjeet yms. Voit myös saada ohjeita Bugzillan
käyttöön IRC-kanavalta #mozillazine palvelimella irc.mozilla.org.

==========================================================================

                          SeaMonkeyn hankkiminen

==========================================================================

Voit ladata SeaMonkeyn yöllisen koosteen mozilla.orgin FTP-sivustolta
osoitteessa

  http://ftp.mozilla.org/pub/mozilla.org/seamonkey/nightly/

Jos haluat aivan viimeisimmän koosteen, katso

  http://www.seamonkey-project.org/dev/nightly

Muista, että yölliset koosteet, joita SeaMonkeyn kehittäjät käyttävät
testaamiseen, saattavat olla epävakaita. Jos etsit viimeistellympää
versiota SeaMonkeystä, SeaMonkey-projekti julkaisee säännöllisesti
julkaisukoosteita, jollaisen voit ladata osoitteesta

  http://www.seamonkey-project.org/releases/

Muista lukea SeaMonkeyn julkaisutiedoista tiedot tunnetuista käyttöön ja
asennukseen liittyvistä ongelmista. Julkaisutiedot löytyvät edellämainitusta
osoitteesta itse julkaisujen kanssa.

==========================================================================

                          Järjestelmävaatimukset

==========================================================================

* Windows

   - Windows 2000, XP, Server 2003, Vista tai 7
   - Intel Pentium -luokan suoritin tai uudempi (500 MHz tai nopeampaa
     suositellaan)
   - 128 Mt keskusmuistia
   - 50 Mt vapaata kiintolevytilaa


==========================================================================

                              Asennusohjeet

==========================================================================

Suositellaan, että suljet kaikki muut ohjelmat ennen asennusohjelman
käynnistämistä. Sinun tulee myös poistaa virustentorjunta käytöstä
väliaikaisesti.

Asenna tyhjään (uuteen) kansioon. Ongelmia voi seurata, jos asennat
aiemmin julkaistun koosteen päälle.

Huom. Näissä ohjeissa ei kerrota, kuinka SeaMonkey käännetään.
Lisätiedot SeaMonkeyn kääntämisestä mozilla.orgin lähdekoodista
saat osoitteesta

  http://developer.mozilla.org/en/docs/Build_Documentation


Asennusohjeet Windowsille
-------------------------

Huom. Jos käytät Windows 2000:ta tai XP:tä, tarvitset järjestelmänvalvojan
oikeudet asentaaksesi SeaMonkeyn. Jos näet asennuksen aikana viestin
"Virhe 5", varmista, että suoritat asennuksen järjestelmänvalvojan
oikeuksilla.


   Tee näin, jos haluat asentaa SeaMonkeyn asennusohjelmalla:

   1. Napsauta linkkiä tiedostoon seamonkey-x.xx.en-US.win32.installer.exe
      (tai vastaavanlainen tiedostonimi) sivustolla, jolta olet lataamassa
      SeaMonkeytä, ladataksesi asennusohjelman koneellesi.

   2. Siirry sijaintiin, johon latasit tiedoston ja kaksoisnapsauta
      asennusohjelman kuvaketta käynnistääksesi asennusohjelman.

   3. Seuraa näytöllä näkyviä ohjeita. SeaMonkey käynnistyy automaattisesti
      ensimmäisellä kerralla.


   Tee näin, jos haluat asentaa SeaMonkeyn .zip-tiedostosta:

   1. Napsauta linkkiä tiedostoon seamonkey-x.xx.en-US.win32.zip
      (tai vastaavanlainen tiedostonimi) sivustolla, jolta olet lataamassa
      SeaMonkeytä, ladataksesi .zip-tiedoston koneellesi.

   2. Siirry sijaintiin, johon latasit tiedoston ja kaksoisnapsauta
      .zip-tiedostoa.

      Huom. Tässä oletetaan, että sinulla on WinZipin
      (tai vastaavan ohjelman) tuore versio valmiiksi asennettuna ja että
      tiedät, miten sitä käytetään. Jos sinulla ei ole sellaista, voit saada
      WinZipin osoitteesta www.winzip.com.

   3. Pura .zip-tiedosto kansioon 
        C:\Program Files\mozilla.org\SeaMonkey.

   4. Siirry kansioon, johon purit SeaMonkeyn ja kaksoisnapsauta
      seamonkey.exe -kuvaketta käynnistääksesi SeaMonkeyn.
